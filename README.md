This project focuses on rebuilding a multi-sheet Excel-based financial model using Python to enable scalable, automated, and reproducible financial analysis. It processes raw financial statements, extracts key metrics, and visualizes company performance over time.

Features
- Automated data extraction from multi-sheet Excel files
- Cleaning and structuring of semi-structured financial data
- Analysis of key financial metrics (Revenue, Gross Profit, Operating Profit)
- Time-series visualization of company performance

Tech Stack
- Python
- pandas – data manipulation
- matplotlib / seaborn – data visualization
- Jupyter Notebook

Key Insights
Identifies trends in revenue and profitability over time
Highlights relationships between key financial metrics
Demonstrates how raw financial data can be transformed into actionable insights

How to Run
Clone the repository
Install required libraries:
pip install pandas matplotlib seaborn
Open the Jupyter Notebook:
jupyter notebook
Run all cells in Financial_Modelling_Project.ipynb

Purpose
This project demonstrates the use of Python for financial data analysis and showcases how traditional Excel-based models can be transformed into more efficient and scalable analytical workflows
